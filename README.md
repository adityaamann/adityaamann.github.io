# Aditya Aman — Personal Portfolio

A clean, minimal personal portfolio website built with HTML, CSS, and JavaScript. Deployed on GitHub Pages.

🌐 **Live Site:** [adityaaman.github.io](https://adityaamann.github.io)

---

## Sections

- **Hero** — Introduction and CTA
- **About** — Bio, stats, and skills
- **Projects** — BuildOrNot, Portfolio Website
- **Certificates** — NPTEL, Oracle, NamasteDev, MUJ
- **Contact** — Email, LinkedIn, GitHub

---

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (DM Serif Display + DM Sans)

---

## Features

- Fully responsive — works on mobile, tablet, and desktop
- Smooth scroll-based reveal animations
- Active nav link highlighting on scroll
- No frameworks, no dependencies — pure HTML/CSS/JS

---

## Run Locally

No setup needed. Just open the file in your browser:

```bash
git clone https://github.com/adityaaman/adityaaman.github.io
cd adityaaman.github.io
open index.html
```

---

## Deployment

Deployed via **GitHub Pages** from the `main` branch.
